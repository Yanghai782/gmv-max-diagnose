# GMV MAX 放量期诊断

冷启跑出了好素材，日订单 5-30 单，ROI 开始稳定——怎么安全放大？

**来源**：壹木行万里 第18/26/54/55/10/11/12/13/23集

---

## 进入放量期的信号

- 日订单稳定在 5-30 单
- 至少 1 条素材日耗 $30+ 且 ROI 达标
- CPA 开始趋于稳定（冷启期的剧烈波动已经过去）
- 连续 2 天预算花 80% 以上

---

## 放量三步走

### 1. 保留好素材（不动原计划）

好素材留在原计划不动，GMV MAX 会自动给它更多预算。不要拆出去单独开计划。

### 2. 逐步加预算

- 每次加价幅度：**20-30%**
- 频率：**每 2 天最多加一次**
- **绝对禁止翻倍**（会让系统重新学习）
- 操作路径：计划 → 编辑 → 日预算 → 改数值 → 保存

### 3. 持续喂新素材

- 频率：**每 3 天加 2-3 条新素材**
- 每计划结构：1-2 条验证过的好素材 + 2-3 条新素材
- GMV MAX 内部赛马：好新素材自动拿预算，差的自然淘汰
- 跑量素材能带动同计划新素材（带动效应）

---

## 花钱太快/太慢怎么调？

### 花钱太慢（预算花不完）

信号：ROI 高但消耗慢 → 目标 ROI 绑太紧

**操作**：降目标 ROI 5-10% 松绑 → 观察 2 天
- 路径：计划 → 编辑 → 目标 ROI → 下调 → 保存

### 花钱太快（ROI < 保本）

信号：消耗快但 ROI 跌破保本线

**操作**：提目标 ROI 5-10% 收紧 → 观察 2 天
- 路径：计划 → 编辑 → 目标 ROI → 上调 → 保存

### 加预算的准确信号（三满足）

1. ROI 连续 2 天达标
2. 当天预算花 80% 以上
3. CPA 稳定（不忽高忽低）

三项全部满足 → 加预算 20-30%。

---

## CPM 飙升怎么办？

### 根因

CTR 和 CVR 不够 → 系统觉得素材转化差 → 需要高价抢流量 → CPM 涨

### 排查顺序

1. **优化素材**（最有效）：换前 3 秒 / 换封面 / 换文案
2. **优化落地页**：检查加载速度、评价数量
3. **检查是否品类整体 CPM 涨**：看大盘是不是竞争加剧

### 不要做的事

- ❌ 降目标 ROI 凑 CPM 高 → 治标不治本且更亏
- ❌ CPM 低也可能是陷阱：便宜流量但人群不对 → 查 CVR 是否远低于平时

---

## 单量卡住怎么办？

### 公式

```
单量 = 消耗 × ROI / 客单价
```

两条路：
- 让系统花更多（加预算 / 降目标 ROI 松绑）
- 提高 ROI（优化素材 CTR/CVR）

### 核心操作：算 GPM 排名

1. 素材报表 → 导出
2. 加列：`GPM = GMV / 曝光 × 1000`
3. 关掉 GPM 最低的素材 → 换 2-3 条新的
4. **每 3 天踢掉表现最差的，每 3 天补 2-3 条新的**

---

## 最大投放模式注意事项

### 素材消耗不均衡是正常的

- 最大投放下，素材消耗差距大 = GMV MAX 内部赛马机制在起作用
- 历史转化好的优先拿预算，冷启过的比新的优先

### 最大投放预算公式

```
最大投放预算 = 目标 ROI 模式预算 × 60-70%
```

太高 = 激进出价 = 容易空耗。

---

## 加热帮素材加速

### 用法

TikTok → 选中 GMV MAX 里好素材 → 加热 → $10-20 跑 1-2 天 → 看数据

### 判定

- 加热后 CTR > 1.5% → 放回 GMV MAX，系统见社交证明更愿给量
- 加热后 CTR < 1% → GMV MAX 里也跑不好 → 换掉

### 注意

- 加热预算不一定能花完，花不完 = 素材不吸引人
- 加热可能干扰原计划 ROI 统计 → 建议新建计划跑加热过的素材
- 凌晨 0-2 点不改设置

---

## 放量期避坑

1. **不要一次翻倍预算** → 系统重新学习
2. **不要只靠 1 条素材撑** → 素材疲劳就没后手
3. **不要因为一天 ROI 差就关** → 给系统 2-3 天观察
4. **不要所有素材都是全新的** → 老带新才是最快路径
5. **不要凌晨改设置** → 该时段数据差没有参考意义
