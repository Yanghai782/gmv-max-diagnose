# GMV MAX 成熟稳定期诊断

日订单 30+ 单，ROI 稳定达标，多条素材在跑——管好素材库是核心。

**来源**：壹木行万里 第22/25/16/13集

---

## 成熟期的标志

- 日订单 30+ 单
- ROI 稳定在保本线以上
- 至少 3 条素材各自日耗 > $30
- CPA 稳定
- 计划跑了至少 7 天以上

达不到这些标准 → 可能还在冷启/放量期。

---

## 真正跑量素材的标准

| 市场 | 日耗标准 | 备注 |
|------|---------|------|
| 东南亚 | 日耗 $50+ 且 ROI 达标 | 低于此不算跑量素材 |
| 美区 | 日耗 $100+ 且 ROI 达标 | 美区流量贵，门槛更高 |

**别培养 2-3 单的素材当主力。** GMV MAX 内部赛马，真正的好马会自己跑出来。

---

## ABCD 素材分类法

按 **花费（高/低）** × **ROI（高于保本/低于保本）** 分为四类：

### 分类矩阵

| | 花费高 | 花费低 |
|---|---|---|
| **ROI ≥ 保本** | **A 类**：保留 + 可加预算 | **C 类**：拉出加热测试 |
| **ROI < 保本** | **B 类**：优化素材后保留 | **D 类**：直接关停 |

### 花费高/低的分界线

用**花费中位数**：所有素材排序后取中间值，高于中位数 = 花费高。

### 各类动作

- **A 类（耗多 + ROI 高）**：保留，可适当加预算。这是你的主力素材，保护好
- **B 类（耗多 + ROI 低/一般）**：保留但需优化。换前 3 秒/封面/CTA，CTR < 1% 优先动前 3 秒
- **C 类（耗少 + ROI 高）**：潜力股。拉出来加热测试（$10-20 跑 1-2 天），CTR > 1.5% 后放回 GMV MAX
- **D 类（耗少 + ROI 差）**：直接关停。不内耗预算

---

## 剔除标准

满足以下 **任一条件** → 直接关停：

| 条件 | 阈值 | 说明 |
|------|------|------|
| CTR 过低 | CTR < 1.0% | 素材不吸引人 |
| CVR 过低 | CVR < 1.5% | 落地页或价格问题（先查落地页再定） |
| 零消耗 | 花费 < $0.5 + 订单数 = 0 | 系统根本不给量 |
| 长期零消耗 | 3 天零消耗 | 系统已放弃该素材 |
| ROI 低于保本 | ROI < 保本 ROI | 降预算观察 2 天，仍不恢复 → 剔除 |

### 清理节奏

- **每 3 天清理一次**
- **每关一条立刻补一条新的**
- 素材生命周期 7-14 天（GMV MAX），到期就换

---

## 老计划加新素材跑不动？

### 原因

新素材跑不动取决于：
- 新素材本身质量（CTR/CVR 是否达标）
- 目标 ROI 是否太紧
- 老素材是否挤占预算（老马把预算吃光了）

### 解决方案

1. **加热加速**：TikTok → 选中视频 → 加热 → $10-20 跑 1-2 天 → CTR > 1.5% 后放回 GMV MAX
2. **加热后 CTR < 1% → 放弃**
3. **如果老计划太挤**：验证过的新素材单独建 GMV MAX 计划，给更大预算
4. **注意**：加热可能干扰原计划 ROI 统计 → 建议新建计划跑加热过的素材

---

## GPM 排名法

每条素材算：`GPM = GMV / 曝光 × 1000`

按 GPM 从高到低排 → 关掉 GPM 最低的 → 换新的。

GPM 比 ROI 更能反映素材"赚钱效率"——花费少的素材 ROI 容易虚高，GPM 才能公平对比。

---

## 成熟期每日检查清单

- [ ] 消耗是否正常？（花不出去 = 有问题）
- [ ] 有没有出单？（连续 3 天 0 单 = 换素材）
- [ ] 哪条素材在跑量？（主力素材要保护好，别乱动）
- [ ] CPM 是否超过 CPM 上限？（超过 = 亏钱，立即换素材）
- [ ] 新素材上线了吗？（至少 3 天加一批）
- [ ] 素材生命周期到了吗？（7-14 天，超期素材即使数据好也要准备替补）
- [ ] 有没有频繁调价/关停？（成熟期最大的坑还是手贱）
