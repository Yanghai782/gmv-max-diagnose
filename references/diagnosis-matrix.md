## 核心指标定义

### 3秒完播率 (3-Second View Rate / Hook Rate)
- **定义**：用户观看素材超过 3 秒的比例
- **合格线**：>25%（北美/欧洲）；>20%（东南亚）
- **优秀线**：>35%
- **核心归因**：开头钩子质量、前 3 秒画面吸引力、人群精准度
- **低 3 秒完播的直接原因**：
  - 开头画面平淡，无视觉冲击
  - 钩子文案不痛不痒，没有击中用户痛点
  - 画面模糊/低质量，用户本能划走
  - 人群包过宽，推给了非目标用户
  - BGM 节奏太慢，开头无强节奏点

### 点击率 (CTR - Click-Through Rate)
- **定义**：看到素材后点击进入落地页/商品详情页的比例
- **合格线**：>1.0%（信息流）；>0.8%（Spark Ads）
- **优秀线**：>2.0%
- **核心归因**：封面/缩略图、文案卖点清晰度、行动号召（CTA）力度
- **低 CTR 的直接原因**：
  - 素材封面/缩略图不够吸引点击
  - 文案没有说清楚"这是什么/为什么要点"
  - CTA 太弱或缺失（没有"立刻下单""点击查看"等引导）
  - 素材表达的东西和用户预期不一致（用户看了但不想深入了解）
  - 素材形式不适合该品类的转化路径

### 转化率 (CVR - Conversion Rate)
- **定义**：点击后完成购买/加购的比例
- **合格线**：>2.0%（标品）；>1.5%（高客单）
- **优秀线**：>5.0%
- **核心归因**：落地页质量、商品详情页、价格竞争力、素材与商品一致性
- **低 CVR 的直接原因**：
  - 素材展示的产品与实际落地页不符（货不对板）
  - 价格在落地页才暴露，超出用户心理预期
  - 落地页加载慢、体验差
  - 商品详情页缺乏信任信号（评价少、无视频评论）
  - 素材激发的兴趣 → 落地页承接不住（素材做得太夸张）

### CPM (千次展示成本)
- **定义**：每 1000 次展示的广告成本
- **合格线**：< （北美）；< （东南亚）
- **核心归因**：人群竞争度、素材质量分（Quality Score）、账户权重
- **高 CPM 的直接原因**：
  - 人群包太窄，竞争激烈
  - 素材质量分低（互动差、负反馈多）
  - 投放时段与大盘竞争期撞车
  - 新品/新账户，系统尚在探索期

### 互动率 (Engagement Rate)
- **定义**：点赞+评论+分享+收藏 / 展示量
- **合格线**：>3%
- **核心归因**：素材共鸣感、话题性、评论区运营
- **低互动率**：素材缺乏让人"想参与"的点
- **高互动但低转化**：素材有趣但与产品关联弱，或驱动的是无效互动

### ROAS / GMV ROI
- **定义**：GMV / 广告花费
- **合格线**：>2.0（盈亏平衡因品类而异）
- **GMV Max 特有**：系统以 GMV 最大化为目标，ROAS 会在学习期波动

## 单指标诊断

### 仅 3 秒完播率低（CTR 和 CVR 正常）
> **用户停留意愿弱，但点击和购买转化没问题**
- 前 3 秒画面未抓住注意力，但一旦用户看完内容，素材质量和转化力是 OK 的
- **诊断方向**：
  1. 开头钩子需要重构（见 creative-optimization.md 钩子公式）
  2. 开头时长节奏检查：信息是否来得太慢？
  3. 检查 BGM 开头前 1 秒是否有强节奏点
  4. 人群包适当收窄，优先推送给高意向用户

### 仅 CTR 低（3 秒完播和 CVR 正常）
> **用户看了但不想点**
- 素材内容本身有趣/好看（3 秒完播好），但缺少驱动点击的动力
- **诊断方向**：
  1. 封面/缩略图是否直接展示了产品利益点？
  2. 文案中 CTA 是否明确？是否有"限时""限量"等紧迫感？
  3. 卖点在素材中是否清晰可见？用户是否清楚"点进去能得到什么"？
  4. 素材中是否展示了价格优势或赠品利益？

### 仅 CVR 低（3 秒完播和 CTR 正常）
> **用户被吸引点击了，但没买**
- 素材和投放没问题，问题在落地转化链路
- **诊断方向**：
  1. 素材卖点与落地页是否匹配？（货不对版是最大元凶）
  2. 落地页加载速度
  3. 商品评价数和评分
  4. 价格是否有竞争力？与素材中的暗示是否一致？
  5. 是否有信任信号（退货政策、包邮、认证标识）

### 仅 CPM 高（其他指标正常）
> **流量成本贵，但素材表现没问题**
- **诊断方向**：
  1. 人群竞争度：是否在该品类高峰期投放？
  2. 素材质量分：是否有隐藏的负反馈信号（用户点了"不感兴趣"）？
  3. 账户消耗历史：是否为新账户？老账户是否被限流？
  4. 出价策略：GMV Max 下的最低 ROI 目标设置是否过于激进？

## 双指标交叉诊断

### 高 3 秒完播 + 低 CTR
> **用户停留了，但不想点击 → 素材有趣但卖点没打出去**
- 这是最经典的"好看不卖货"型素材
- **归因**：内容型素材（搞笑、剧情、颜值类开头）做足了停留但没转化
- **优化方向**：
  1. 在前 5 秒内插入产品利益点（在用户还没划走前种下购买动机）
  2. 检查 CTA 出现的时机：是否太晚？建议在第 5-8 秒第一次出现 CTA
  3. 封面改为产品效果图 + 价格锚点
  4. 文案测试：从软性表达改为直接的利益承诺
  - **注意**：此场景不要大改开头（3 秒完播是你最大的资产），而是优化内容中段和 CTA

### 低 3 秒完播 + 高 CTR
> **开头没留住人，但留下来的人想点 → 封面党/标题党**
- 封面或文案很吸引人，但素材内容撑不住开头的期待
- **诊断方向**：
  1. 开头的画面/信息是否与封面一致？用户是否感到"被骗了"？
  2. 前 3 秒内容是否过于平淡（和封面强烈的利益承诺形成落差）？
  3. 钩子方法：把封面的卖点用视觉化方式在前 1-2 秒就展示出来
  - **注意**：这个组合如果不改，会拉低素材质量分，最终导致 CPM 上升、量级下降

### 高 CTR + 低 CVR
> **点击欲望强但不下单 → 落地页问题或素材货不对板**
- 素材的吸引力到了落地页就断裂了
- **诊断方向**：
  1. 素材展示的产品与落地页产品是否一致？（配色、型号、功能描述）
  2. 素材暗示的价格与实际价格是否有差距？
  3. 落地页首屏是否直接展示了素材中提到的核心卖点？
  4. 评价数量：少于 50 条评价时，CVR 天然偏低
  5. 是否有买家秀视频/图片评论？

### 低 CTR + 高 CVR
> **点的人少但点了就买 → 精准但人群太小**
- 素材精准地打中了核心购买人群，但覆盖面不够
- **诊断方向**：
  1. 人群包是否需要适当放宽？（相似人群、兴趣扩展）
  2. 素材风格是否过于垂直？（例：只吸引特定小圈子用户）
  3. 可以考虑做多套不同风格的素材，覆盖更宽的受众人群
  4. 这个素材适合作为"精准收割"素材保留，同时另做拉新素材

### 高 3 秒完播 + 低 CVR
> **看的人多但买的人少 → 内容有趣但与产品脱节**
- 娱乐型素材的通病
- **诊断方向**：
  1. 素材中产品出现的时机和方式：是否过于边缘化？
  2. 产品卖点在素材中是否被娱乐内容覆盖了？
  3. 可以尝试"功能展示型"素材替代纯娱乐型
  4. 检查人群：是否因为内容有趣而吸引了大量非目标用户？

### 低完播 + 低 CTR（双低）
> **全面翻车 → 从头重建**
- 素材从开头到封面都不行
- **诊断方向**：
  1. 先检查人群是否方向正确（可能完全打错了人群）
  2. 检查素材基础质量：画面清晰度、BGM、字幕、口播是否专业
  3. 对标竞品素材，找差距
  4. 建议：从 0 开始，先做 5-10 条不同钩子的素材同时测试

### 高 CPM + 低 CTR
> **花钱多还点得少 → 素材质量分低，系统不愿给量**
- 系统判定素材不受欢迎
- **诊断方向**：
  1. 素材是否有用户负反馈（不感兴趣/举报）？
  2. 素材是否与其他大量投放的素材高度雷同？
  3. 尝试更换素材形式（视频→直播切片，混剪→实拍）
  4. 暂停低效素材，让账户"冷却"后再上新素材

### 高 CPM + 高 CTR + 低 CVR
> **竞争激烈品类，点击贵，转化还差**
- 说明品类本身 CPM 高（竞争大），但素材和落地页也都没做好
- **诊断方向**：
  1. 先优化落地页和评价（把 CVR 拉上来）
  2. 然后通过素材差异化降低 CPM（独特素材 ≠ 模板混剪）
  3. 考虑非高峰时段投放降低成本

## 综合诊断

### 场景 A：高 CTR + 高 CVR + 量级上不去
> **素材质量好，但花钱花不出去 / 消耗不增长**
这是 GMV Max 最常见的"天花板"场景：
- **归因**：
  1. **人群天花板**：当前受众池已经被充分触达，新增用户不足
  2. **预算限制**：日预算太低，系统不敢放量
  3. **出价/ROI 目标过紧**：最低 ROI 设置太高，系统认为不划算
  4. **素材疲劳**：素材到了衰退期（通常 7-14 天）
- **解决路径**：
  1. **扩人群**：从精准兴趣→相似人群→宽兴趣，逐层扩展
  2. **提预算**：日预算提高 30-50%，观察 24-48 小时
  3. **松 ROI 目标**：将 GMV Max 的最低 ROI 下调 10-20%
  4. **上新素材**：至少每周 3-5 条新素材，保持素材健康度
  5. **多账户分流**：如果单账户天花板明显，考虑多账户同时跑

### 场景 B：低 CTR + 低 CVR + 低完播（三低）
> **全维度翻车，需要系统性重建**
- **归因**：可能人群方向完全错误，或素材质量严重不达标
- **解决路径**：
  1. **先确认人群方向**：重新分析目标用户画像
  2. **对标竞品**：用 BigSpy/Pipiads 等工具找同类目 TOP 素材
  3. **重建素材策略**：从 5 条不同风格的素材同时起跑
  4. **冷启动方法**：见 cold-start.md

### 场景 C：ROI 稳步下降（素材疲劳信号）
> **数据慢慢变差，不是骤降**
- **疲劳指标**：
  - CTR 每周下降 >10%
  - CPM 每周上升 >15%
  - 3 秒完播率缓慢下降（用户开始看腻）
  - 频次（Frequency）>3
- **解决路径**：
  1. 判断疲劳阶段：轻度疲劳（降量 20%）→ 中度（降量 50%）→ 重度（停素材）
  2. 换血节奏：主力素材 7-10 天一个迭代周期
  3. 微迭代 > 大换血：在成功素材基础上改开头/改 BGM/改 CTA，而非完全重做

### 场景 D：3 秒完播正常 + CTR 正常 + CVR 突然跌
> **指标突然恶化 → 外部因素**
- **归因**：
  1. 竞品降价/促销活动
  2. 商品出现差评（检查近期评价）
  3. 落地页出技术问题
  4. 大盘流量变化（节假日、大促前后）
- **检查顺序**：先查评价 → 再查落地页 → 再看竞品 → 最后看大盘

### 场景 E：流量大但 ROI 低（典型跑量不跑效）
> **花钱如流水，GMV 起不来**
- **归因**：
  1. GMV Max 学习期出价过于激进
  2. 人群太宽，大量触达了非购买意图用户
  3. 素材偏娱乐/泛流量型，吸引的是"围观群众"
- **解决路径**：
  1. 上调 GMV Max 最低 ROI 目标 20-30%
  2. 收窄人群定向
  3. 替换为转化导向型素材（功能展示、使用场景、痛点解决）
