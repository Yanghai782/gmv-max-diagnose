# GMV MAX 核心公式库

基于博主 壹木行万里 第29/30/32/55集内容。

---

## 1. 保本 ROI

```
保本 ROI = 客单价 / 利润
```

- **客单价**：商品售价
- **利润**：售价 - 产品成本 - 物流 - 佣金 - 退货损耗
- **GMV MAX 定价必须考虑广告费**：目标售价 - (目标售价 / 目标 ROI) - 物流 - 佣金 - 退货 = 利润
- 保本 ROI 是 GMV MAX 所有操作的底线，低于保本就亏

**示例**：
- 售价 $39.99，成本 $12，物流 $5，佣金 $4（~10%），退货率 5%（$2）
- 利润 = 39.99 - 12 - 5 - 4 - 2 = $16.99
- 保本 ROI = 39.99 / 16.99 ≈ 2.35

---

## 2. GPM（千次展示产出）

```
GPM = GMV / 曝光 × 1000
```

- 衡量每千次展示带来的销售额，素材的核心产出指标
- **GPM = CTR × CVR × 客单价 × 1000**
- 用法：素材报表导出 → 加列 GPM → 按 GPM 排名 → 关掉 GPM 低的换新的
- GPM 比 ROI 更能反映素材的"赚钱效率"

---

## 3. CPM 上限

```
CPM上限 = 客单价 × CTR × CVR × 1000 / 保本ROI
```

- 实际 CPM > CPM 上限 → 素材在亏钱，应该淘汰
- 实际 CPM < CPM 上限 → 素材可以继续跑
- **这是 GMV MAX 最硬的一条淘汰标准**

**推导过程**：
- ROI = 总收入 / 广告成本
- 总收入 = 客单价 × 订单数 = 客单价 × CVR × 点击量
- CPM = 花费 / 曝光 × 1000
- 保本 ROI 时：GMV / 花费 = 保本 ROI
- 代入得：CPM上限 = 客单价 × CTR × CVR × 1000 / 保本 ROI

**注意 CTR 和 CVR 代入公式时要用小数**（如 CTR=1.2% 则用 0.012）

---

## 4. ECPM（千次展示收益）

```
ECPM = CTR × CVR × 客单价 × 1000
```

- ECPM 是系统分配流量的核心指标，决定素材竞争力
- 低客单 + 合理 ROI = 更高转化率 → ECPM 更高 → 系统更优先投放
- **素材好坏不看你觉得好不好看，看 ECPM**

---

## 5. 转化率（曝光级）

```
转化率(曝光级) = 转化数 / 曝光 × 100%
```

- 与 CVR 区分：CVR = 转化数 / 点击数
- 曝光级转化率用于不同素材之间的公平对比（排除点击量差异）
- 和 GPM 一起看：GPM 高 + 转化率高 = 真正的好素材

---

## 6. CPA（每转化成本）

```
CPA = 广告花费 / 转化数
```

- 冷启期 CPA 从高到低慢慢稳
- 成熟期 CPA 应该稳定，持续上升是衰退信号
- 新计划出价参考：目标 CPA × 1.2 ~ 1.5

---

## 7. 素材对比公平公式

统一以 1000 曝光为基数算 GPM 和转化率，公平对比不同花费水平的素材：

```
GPM = GMV / 曝光 × 1000
转化率(曝光级) = 转化数 / 曝光 × 100%
```

**为什么不用 ROI？** ROI 受花费影响大，花费少的素材 ROI 容易虚高，GPM 才能真实反映素材产出效率。
