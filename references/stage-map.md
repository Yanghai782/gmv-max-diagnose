# GMV MAX 阶段导航地图

打开 TikTok Ads 后台 → GMV MAX → 看昨天 GMV / 订单数 / ROI → 对号入座。

---

## 第0章：GMV MAX 基础认知

**GMV MAX 是什么？**

GMV MAX 是 TikTok 电商广告的自动化投放模式。你只管上传素材 + 设目标 ROI + 给预算，系统自动找人、分配预算、跨版位赛马。

**三种模式怎么选？**

| 模式 | 用途 | 何时用 |
|------|------|--------|
| **目标 ROI** | 稳定盈利跑量 | 已知保本 ROI，想稳定出单 |
| **最大转化量** | 快速冲单 | 新品冷启，不设 ROI 约束 |
| **最大投放量** | 公平测素材 | 批量上新素材，看哪条能跑 |

**实操路径**：新品先跑 2-3 天最大投放量测素材 → 测出好素材切最大转化量冲单 → 稳定后切目标 ROI 长线跑。

**GMV MAX 最该盯的四个指标（按重要性）：**

1. **消耗**：花没花出去？花不动 = 目标 ROI 太高或素材差
2. **ROI**：跟保本线比，达标继续，不达标排查
3. **CTR**：素材吸不吸引人，CTR < 1% 换素材
4. **CVR**：落地页行不行，CVR 突然降 = 落地页或价格出问题

**GMV MAX 不需要管的**：人群定向（自动）、分时段出价（自动）、版位分配（自动跨版位）。

## 阶段判定速查

| 你的数据 | 对应阶段 | 翻到 |
|---------|---------|------|
| 新品刚上架，日订单 0-5 单，ROI 不稳定 | **第1章：冷启期** | `cold-start.md` |
| 跑出了好素材，日订单 5-30 单，想放大 | **第2章：放量期** | `scaling.md` |
| 日订单 30+，ROI 稳定达标，多条素材在跑 | **第3章：成熟稳定期** | `maturity.md` |
| ROI 连降 3 天，换素材也没用 | **第4章：衰退期** | `decline.md` |

---

## 各阶段核心原则

### 冷启期 → 不看 ROI
系统从 0 学习 素材+产品+买家 三者匹配。冷启 3-7 天，CPA 从高到低慢慢稳。只看 CTR + CVR + 消耗速度。

### 放量期 → 控制节奏
保留好素材 + 逐步加预算（每次 +20-30%，2 天加一次）+ 持续喂新素材（每 3 天加 2-3 条）。

### 成熟期 → 管好素材库
每 3 天清理：ABCD 分类 + 剔除低效素材。GMV MAX 素材生命周期 7-14 天，到期就换。

### 衰退期 → 判断根因
换新素材 ROI 恢复 = 素材疲劳。换多套全新素材 ROI 持续降 = 产品周期到。
