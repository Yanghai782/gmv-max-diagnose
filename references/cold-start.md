# GMV MAX 冷启动期诊断

新品刚上架，日订单 0-5 单，ROI 不稳定——系统从零开始认识你的产品和素材。

**来源**：壹木行万里 第2/3/4/5/6/7/8/9/16/17/21/28集；PPT2 Part 01-04

---

## 冷启动的本质

**GMV MAX 冷启 = 系统从 0 学习 素材 + 产品 + 买家 三者匹配的过程。**

A1（触达）→ A2（点击）→ A3（加购）→ A4（下单）→ A5（复购）

系统不认识你的产品 → 从最广人群开始筛选 → 越跑越精准 → 成本趋于稳定。

---

## 冷启期核心原则

### 最重要的三条

1. **冷启不看 ROI**：冷启的本质是花钱让 GMV MAX 学会，不是追求利润
2. **只看消耗、CTR、CVR**：这三项是冷启期的唯一有效指标
3. **别频繁操作**：每次调价/关停/改定向都打断系统学习 = 永远在冷启

### 冷启时长与预算

- **时长**：3-7 天
- **日预算**：$30-50（太多浪费，太少学不动）
- CPA 从高到低慢慢稳，这个过程是正常的

---

## 三种冷启方法

### 方法一：目标 ROI 模式冷启

**适用**：已知保本 ROI，希望稳扎稳打

**步骤**：
1. 素材：至少 5-10 条视频，风格各异（开箱/测评/场景/痛点）
2. 目标 ROI = 保本 ROI
3. 日预算 $30-50，跑 2-3 天别动
4. 每天看素材报表：
   - CTR > 1.5% → 方向对
   - CTR 1.0-1.5% → 再观察
   - CTR < 1.0% → 换素材
5. 冷启完成信号：连续 2 天 CPA 稳定 + 预算花 80% 以上 + 累计 20-50 转化

### 方法二：加热辅助冷启

**适用**：商品卡抢走预算，素材不花钱

**步骤**：
1. 原因：素材太差，系统退而求其次投商品卡
2. 三步：
   - 调高目标 ROI 收紧预算
   - 加热素材补数据（dou+ 设 $10-20 跑 1-2 天）
   - CTR > 1.5% 后调回原 ROI
3. 加热目的：给素材加社交证明，GMV MAX 更信任

### 方法三：最大投放量测素材

**适用**：目标 ROI 模式几乎不花钱，想快速批量测素材

**步骤**：
1. 切换模式：计划 → 优化目标 → 最大投放量（Max Delivery）
2. 放 5-10 条素材，日预算 $30-50
3. **第一波**跑 1-2 天看 CTR/CVR
4. **第二波**：CTR > 1.5% 的切目标 ROI 模式加预算，差的关掉
5. 最大投放优势：均匀分配预算，公平赛马

### 实操路径推荐

新品先跑 2-3 天最大投放量测素材 → 测出好素材切最大转化量冲单 → 稳定后切目标 ROI 长线跑。

---

## 冷启成功信号

| 指标 | 标准 |
|------|------|
| 出单 | 系统开始稳定出单（日均 >= 5 单） |
| CPA | 在可接受范围内（冷启初高后降） |
| 素材 | 有 1-2 条成为跑量素材 |
| 预算消耗 | 当天预算花 80% 以上 |
| 转化量 | 累计 20-50 转化 |

---

## 冷启期常见问题

### 瞬烧（几分钟花完预算）

**原因**：系统觉得素材高潜力给大量流量，但转化没跟上

**处理**：
- 24 小时内有转化 → 别动，系统自调
- 零转化 → 关计划换素材重建

**预防**：
- 选标准投放（不加速）
- 日预算 <= $50
- 前 3 天不调任何设置

### 商品卡抢预算

素材太差 → 系统退而求其次投商品卡。

**处理**：调高目标 ROI 收紧 → 加热素材补数据 → CTR 好转后调回。

### 消耗慢/花不动

- 目标 ROI 太高 → 降 5-10% 松绑
- 素材不够/太差 → 加到 5-10 条
- 人群太窄 → GMV MAX 自动探索，不需要手动调

### 消耗忽高忽低、CPA 忽高忽低

**正常现象**！GMV MAX 自动从宽到窄探索，这是系统在学习。你只管素材质量和预算，数据波动前 3-7 天都正常。

---

## 影响冷启速度的因素（按重要性）

1. **素材质量** > 2. **目标 ROI 松紧** > 3. **预算大小** > 4. **品类竞争**

---

## 冷启新建时机

- ✅ 连续 3 天 ROI 降 > 15% → 新建
- ✅ 瞬烧后 24 小时零转化 → 关掉新建
- ❌ 一天数据差 → 不要新建
- ❌ 同素材微调 → 不要新建

---

## 冷启期每日检查清单

- [ ] 消耗是否正常？（花不出去 = 目标 ROI 太高或素材差）
- [ ] 有没有出单？（连续 3 天 0 单 = 换素材方向）
- [ ] CTR 是否 > 1.0%？（< 1.0% 立刻换素材）
- [ ] CVR 是否 > 1.5%？（< 1.5% 查落地页）
- [ ] 素材数量够吗？（至少 5-10 条）
- [ ] 有没有频繁调价/关停？（每操作一次系统重新学一次）
- [ ] 冷启成功信号出现了吗？（日均 >= 5 单 + CPA 稳定）

---

## 冷启期最大禁忌

**频繁调价、关停、改定向 = 每操作一次，系统重新学一次 = 永远在冷启，永远跑不起来。**

大多数 TikTok 广告跑不起来的原因不是产品不行，是投手太勤快了。
